* 내, 외부 네트워크 구간 사이에 위치한 중간 지점

* 접근 제한을 수행
* 외부 네트워크에서 직접 접근이 가능
* 설명
  * 회사에서는 서비스를 제공하는 서버와 내부에서만 동작해야하는 서버가 있다.
  * 외부 접근 가능: 웹서버, DNS 서버, 메일 서버
  * 외부 접근 불가능: DB 서버
  * 외부에서 접근이 가능한 서버를 DMZ에 두고, 보호해야 하는 것을 내부 네트워크에 두어, 외부 침입을 제한하는 역할
